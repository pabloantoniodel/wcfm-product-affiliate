Sistema de afiliación de productos para WCFM Marketplace sin clonación, con tracking y comisiones duales.

== Description ==

WCFM Product Affiliate transforma tu marketplace WCFM en un sistema de afiliación donde los vendedores pueden vender productos de otros sin clonarlos.

= Características Principales =

* Sin clonación de productos
* Tracking automático de origen de ventas
* Sistema de comisiones duales (dueño + afiliado)
* Dashboard completo para vendedores
* Reportes detallados para administradores
* Integración total con WCFM Marketplace
* Fácil de usar con botones simples

= Cómo Funciona =

1. El vendedor añade productos de otros a su catálogo de afiliados
2. Los clientes compran desde su tienda
3. El pedido va al dueño del producto
4. Se registra el origen de la venta
5. Ambos vendedores reciben comisión automáticamente

= Beneficios =

* Para vendedores: Vende más productos sin inventario
* Para dueños: Aumenta ventas con red de afiliados
* Para clientes: Más opciones en cada tienda
* Para admins: Sistema de comisiones automatizado

== Installation ==

1. Sube la carpeta del plugin a `/wp-content/plugins/`
2. Activa el plugin en el menú 'Plugins' de WordPress
3. Ve a WCFM > Settings > Marketplace > Product Affiliate
4. Configura las opciones según tus necesidades

= Requisitos =

* WordPress 5.0 o superior
* WooCommerce 3.0 o superior
* WCFM Frontend Manager 6.0 o superior
* WCFM Marketplace 3.0 o superior

== Frequently Asked Questions ==

= ¿Se clonan los productos? =

No. Los productos permanecen en la cuenta del dueño original. Solo se crea una relación de afiliación.

= ¿Quién gestiona el pedido? =

El dueño del producto recibe y gestiona el pedido completo, incluyendo envío y atención al cliente.

= ¿Cómo se calculan las comisiones? =

Por defecto, el afiliado recibe 20% y el dueño 80%. Esto es configurable en ajustes.

= ¿Cuándo se pagan las comisiones? =

Las comisiones se procesan cuando el pedido alcanza estado "Completado" o "Procesando".

= ¿Es compatible con otros plugins? =

Está diseñado específicamente para WCFM Marketplace. La compatibilidad con otros plugins depende de cada caso.
